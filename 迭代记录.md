## 2026-04-20 - Day 7 迭代

### 迭代内容

**目标**：基于 Day 2 学到的 JavaScript 工具函数抽取、Day 5 学到的 CSS 动画，增强搜索体验和交互流畅度

### 具体改动

1. **抽取工具函数（代码规范）**
   - `debounce(fn, delay)` — 防抖函数，将高频触发合并为最后一次执行
   - `highlightText(text, keyword)` — 搜索关键词高亮，正则转义防止 XSS
   - `animateNumber(element, target, duration)` — 数字变化时平滑过渡，使用 easeOut 缓动

2. **搜索关键词高亮**
   - 卡片名称中匹配搜索词的部分以青绿色高亮显示
   - 使用 `<mark class="search-highlight">` 语义化标签
   - 正则转义保证特殊字符安全

3. **结果计数动画**
   - 筛选结果变化时，数字带缩放 + easeOut 动画过渡
   - `.filtered-count` 新增 CSS 过渡效果

4. **空结果友好提示**
   - 显示当前搜索关键词（引号包裹）
   - 根据是否有搜索词给出不同建议文案
   - 有搜索词时显示"清除搜索"按钮，无时显示"清除筛选"

5. **点赞按钮动画优化**
   - 数字从旧值平滑过渡到新值（300ms easeOut）
   - 点击时按钮脉冲放大（scale 1.3）再回弹
   - 替代原有的简单 setTimeout 心跳动画

### 技术细节

- 纯原生 JS 实现，无新依赖
- 防抖延迟从 300ms 优化到 250ms，更跟手
- highlightText 使用正则转义，防止搜索特殊字符导致问题
- animateNumber 使用 `performance.now()` + `requestAnimationFrame`，性能更优

### 文件变更

- `Agent-Street/index.html` - 工具函数区 + CSS 样式 + renderAgentCard + applyFilters + toggleLike + renderNoResults + setupSearchListeners
