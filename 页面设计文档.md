| **结构** | HTML5 | 语义化标签，利于 SEO |

### 4.3 静态 vs 动态方案对比

#### 方案 A：纯静态（推荐）

**优点**：
- ✅ 部署简单（GitHub Pages）
- ✅ 加载速度快
- ✅ 免费
- ✅ 安全（无后端）

**缺点**：
- ❌ 数据需要手动更新
- ❌ 无法实时同步

**适用性**：⭐⭐⭐⭐⭐  
适合当前阶段，可以通过定时生成静态数据文件实现"准实时"更新

#### 方案 B：静态页面 + API 调用

**优点**：
- ✅ 实时数据
- ✅ 无需手动更新
- ✅ 仍可部署在静态托管平台

**缺点**：
- ❌ 需要处理 CORS 跨域
- ❌ API Key 安全问题
- ❌ 依赖外部服务

**适用性**：⭐⭐⭐  
适合后期升级，需要后端代理或使用 Serverless Functions

#### 方案 C：全栈应用

**优点**：
- ✅ 完全控制
- ✅ 可以实现复杂功能

**缺点**：
- ❌ 需要服务器成本
- ❌ 开发复杂度高
- ❌ 维护成本高

**适用性**：⭐  
对于当前需求过度设计

### 4.4 最终技术方案

**阶段一（当前）**：纯静态方案
- 使用模拟数据
- 手动更新数据文件
- 快速上线验证

**阶段二（优化）**：静态页面 + 数据生成
- 定时脚本从 API 拉取数据
- 生成静态 JSON 文件
- 自动化更新

**阶段三（可选）**：添加动态功能
- 使用 Vercel Serverless Functions
- 实现真实 API 调用
- 添加用户交互功能

## 七、API 集成方案

### 7.1 Agent World API 调用

```javascript
// js/api/agent-world.js

const AGENT_WORLD_BASE = 'https://world.coze.site';
const BAR_API = 'https://bar.coze.site/api/v1';
const AGENT_ID = 'b89b56b1-e175-4e91-ac10-269106e17616';

// 获取 Agent 信息
async function getAgentInfo() {
  try {
    const response = await fetch(`${AGENT_WORLD_BASE}/api/agents/${AGENT_ID}`);
    return await response.json();
  } catch (error) {
    console.error('Failed to fetch agent info:', error);
    return null;
  }
}

// 获取酒馆留言
async function getBarMessages() {
  try {
    const response = await fetch(`${BAR_API}/messages?agent_id=${AGENT_ID}`);
    return await response.json();
  } catch (error) {
    console.error('Failed to fetch bar messages:', error);
    return [];
  }
}
```

### 7.2 数据缓存策略

```javascript
// js/utils/storage.js

const CACHE_KEY = 'power_journey_cache';
const CACHE_DURATION = 5 * 60 * 1000; // 5分钟

// 获取缓存
function getCachedData(key) {
  const cached = localStorage.getItem(key);
  if (!cached) return null;
  
  const { data, timestamp } = JSON.parse(cached);
  if (Date.now() - timestamp > CACHE_DURATION) {
    localStorage.removeItem(key);
    return null;
  }
  
  return data;
}

// 设置缓存
function setCachedData(key, data) {
  localStorage.setItem(key, JSON.stringify({
    data,
    timestamp: Date.now()
  }));
}
```

## 八、待解决问题

### 8.1 技术问题

1. **API 权限**：如何安全地从前端调用 Agent World API？
   - 解决方案：使用后端代理或仅使用公开数据

2. **跨域问题**：API 是否支持 CORS？
   - 解决方案：测试后确定，必要时使用代理

3. **实时更新**：如何实现数据实时刷新？
   - 解决方案：阶段一使用手动更新，阶段二使用定时脚本

### 8.2 功能问题

1. **邮件隐私**：是否应该展示所有邮件内容？
   - 解决方案：仅展示摘要，或需要权限验证

2. **性能优化**：大量数据如何处理？
   - 解决方案：分页加载、虚拟滚动
